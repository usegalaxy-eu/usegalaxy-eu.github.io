ELIXIR Galxy Admin workshop in Oslo.

»
Jan 12, 2018
Freiburg training

We had a very nice Galaxy Admin workshop in Oslo.

#oslo airport nailed it! #usegalaxy pic.twitter.com/vHh1nMehWK

— Björn Grüning (@bjoerngruening) January 13, 2018

Over 30 participants from all over Europe learned how to setup a highly scalable Galaxy server and how to use resource efficiently. We would like to thank the traininers Nicola Soranzo (GB), Marius van den Beek (FR), Enis Afghan (USA), Abdulrahman Azab (NO) and Björn Grüning (DE), as well as our sponsors ELIXIR, de.NBI and the awesome host - the University of Oslo.

Today we delete everything what we have done to set up a #usegalaxy server in the last 2 days and redo it with #ansible in a few minutes. #Oslo Galaxy Admin training. @ELIXIREurope #denbi pic.twitter.com/QigCnUreH4

— Björn Grüning (@bjoerngruening) January 10, 2018
